cemu模拟器畅玩所有WiiU游戏（以旷野之息为例）

第一步 安装代码运行环境

  • 安装python

​

https://www.python.org/
python下载

​ Python为后续工具所使用的编程语言，是工具运作的前提。建议安装 Python 3.8.8 版本，避免出现一些意料之外的问题。

第二步 安装工具

  • 创建文件夹

​ 在你喜好的盘创建文件夹，用以存放所有相关文件

​ 如：F:\WiiU

  • 安装FunKiiU（用于从任天堂官方WiiU商店下载游戏）

​

https://github.com/llakssz/FunKiiU
FunKiiU项目GitHub库

​ 下载后解压至F:\WiiU\tools

  • 安装cdecrypt（用于解码从任天堂官方WiiU商店下载的游戏）

​

https://github.com/VitaSmith/cdecrypt
cdecrypt项目GitHub库

​ 下载后解压至F:\WiiU\tools

  • 安装CEMU（pc端WiiU模拟器，用来运行游戏）

​

https://cemu.info/
cemu项目GitHub库

​ 旷野之息多人联机MOD仅支持 CEMU1.26.2f 版本 ，实测最新版本加载游戏更快，目前CEMU项目已经停止更新

​ 下载后解压至F:\WiiU\CEMU

​ 双击cemu.exe运行，进行模拟器路径的配置

​ 在WiiU中新建“mlc”和“games”两个文件夹

​ 将模拟器对应文件路径指向这两个文件夹

第三步 获取游戏（以旷野之息为例）

1. 找到对应的游戏资源

https://www.wiiuemulator.com/JPN-Game-Key-Database.htm
获取id和key

点击USA/EUR/JPN按钮选择想要下载的游戏版本。

香蕉网上的旷野之息mod多适用于日版，而马里奥赛车8美版可以避免在cemu上的启动问题

“Ctrl+F”打开网页搜索功能，查找想要下载的游戏，每个游戏会有三个同名词条，看右侧type分别是eshop/DLC/Patch，分别是该游戏的本体、扩展包、更新包。

2. 使用FunKiiU工具下载

  1. 打开FunKiiU文件夹（F:\WiiU\tools\FunKiiU）

  2. 右键，选择在终端中打开

  3. 检验python是否安装成功

    输入

    正确输出为

  4. 输入以下Python代码

​ 以同样方式将旷野之息的本体、扩展包、更新包都下载一遍，等待下载完成

3. 使用cdecrypt工具解码

  1. 打开“F:\WiiU\tools\FunKiiU\install”文件夹
  2. 打开”F:\WiiU\tools\cdecrypt\“文件夹
  3. 分别将install文件夹中的三个文件拖动到cdecrypt.exe程序上进行解码，各自的解码结果位于之前三个文件夹内部

第四步 将游戏安装进模拟器

- 安装本体

  1. 在“F:\WiiU\games”文件夹中新建一个文件夹，以游戏名称命名（为避免路径错误，不要有中文）
  2. 将“F:\WiiU\tools\FunKiiU\install”内游戏本体文件夹（00050000101c9300）里面解码出来的文件移动到对应游戏命名的文件夹内

- 安装DLC和更新

  1. 进入”F:\WiiU\mlc\usr\title”文件夹，**“c”结尾的文件夹名称对应下载游戏时DLC文件title id的前半段；“e”**结尾的文件夹名称对应下载游戏时更新包title id的前半段
  2. 查看“F:\WiiU\tools\FunKiiU\install”内DLC文件夹以及Update文件夹，文件夹名称分别为对应的title id
  3. 打开**“c”**结尾的文件夹，在里面新建一个文件夹，命名为DLC文件title id的后半段，两个文件夹的名称刚好可以组成完整的title id，将DLC文件夹中解码出来的文件移动进去
  4. 打开**“e”**结尾的文件夹，在里面新建一个文件夹，命名为Update文件title id的后半段，两个文件夹的名称刚好可以组成完整的title id，将Update文件夹中解码出来的文件移动进去

- 检验

双击cemu.exe打开模拟器，右键，刷新游戏目录，可以看到游戏版本和DLC版本，说明模拟器路径正确，路径内游戏本体、dlc、更新包文件正确

- 模拟器配置

（系统设置成中文）模拟器面板——>选项——>图形插件——>下载社区最新图形插件

网络不好的话手动从图形插件官网下载然后加压到“cemu”文件夹内的“graphicPacks”文件夹中

https://github.com/cemu-project/cemu_graphic_packs
手动下载图形插件
